Lesson 8: Advanced Topics on Pattern Discovery

Lesson Overview

In this lesson, you will learn several advanced topics on pattern discovery including mining frequent patterns in data streams, pattern discovery for software bug mining, pattern discovery for image analysis, and pattern discovery and society: privacy-preserving pattern mining. Finally, we look forward to the future on research and applications of pattern mining.

Time

This lesson should take approximately 2 hours of dedicated time to complete, with its videos and assignments.

Goals and Objectives

After you actively engage in the learning experiences in this module, you should be able to:

  • Understand a broad spectrum of advanced topics on pattern discovery in data mining.

  • Know the basic concepts and techniques for mining frequent patterns in data streams.

  • Understand methods for mining copy-and-paste bugs as an example of pattern discovery for software bug mining.

  • Understand some basic skills of pattern discovery for image analysis.

  • Understand that pattern discovery is closely linked to societal concerns and know some interesting studies on privacy-preserving pattern mining.

  • Understand that research into methods and applications of pattern discovery is still an active frontier in data mining research.

Guiding Questions

Develop your answers to the following guiding questions while watching the video lectures throughout the lesson.

  • Why is it challenging to mine frequent patterns in data streams, even to mine frequent single items? Why can lossy counting ensure the error bound no matter how long the data stream is?

  • Why is pattern discovery useful for mining software bugs? How can a bug mining problem such as copy-and-paste bug mining transform into a pattern mining problem?

  • Why is pattern mining useful for image analysis? Can you give one good example to show pattern discovery may help identify critical components in image analysis?

  • Why do we want to ensure privacy in pattern analysis? Can you present one case in which privacy can be ensured in frequent pattern mining?

  • What are the cool research issues in pattern mining that may have interesting new applications?

Key Phrases/Concepts

Keep your eyes open for the following key terms or phrases as you complete the readings and interact with the lectures. These topics will help you better understand the content in this module.

  • Stream data mining; mining frequent patterns in data streams; the lossy counting algorithm

  • Software bug mining; pattern discovery for software bug mining; mining copy-and-paste bugs

  • Image analysis; pattern mining for image analysis; visual items; visual patterns

  • Three categories on privacy issues in data mining: input privacy, output privacy and owner privacy, k-anonymity and differential privacy; data perturbation for privacy-preserving pattern mining

  • Invisible data mining; new application exploration
